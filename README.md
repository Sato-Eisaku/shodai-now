# #商大なう
![アイコン](docs/images/large-icon.png)

## 概要
大学生活をより楽しむために作ったTwitter連携アプリです．

## URL
https://shodai-now.netlify.app/

## 機能
### ツイートをする
場所に対してつぶやきを残すことができます．
![ツイート画面に行くまで](docs/images/tweet.gif)

### ツイートを見る
みんながどこで何をつぶやいたのかを見ることができます．
![ツイート一覧](docs/images/tweet.png)
